## 1. Project Overview

This project delivered a comprehensive, open-source Cardano dApp development course designed specifically for the Vietnamese developer community. The course guides beginners from basic web development fundamentals through Cardano library integration (MeshJS, Lucid) to smart contract development with Aiken — culminating in building and deploying a full NFT Marketplace on Cardano.

All course materials — including 14 video tutorials, slide decks, instructional articles, source code repositories, and practical exercises — are publicly available through our Learning Management System (LMS) and GitHub.

## 6. Why Is This Project Important to the Cardano Ecosystem?

Vietnam has a large and growing tech talent pool, but the Cardano developer community lacks comprehensive, structured learning resources in Vietnamese. Most existing documentation and tutorials are in English, creating a significant barrier for local developers who want to build on Cardano.

This project directly addresses this gap by providing:

1. **A complete learning path** — from web fundamentals to deploying a full Cardano dApp, eliminating the fragmented learning experience developers previously faced.
2. **Practical, hands-on education** — every lesson includes working code, exercises, and real blockchain interactions, not just theory.
3. **A real project outcome** — students build and deploy an NFT Marketplace, demonstrating tangible Cardano development skills.
4. **Sustainable community growth** — by training local developers who can mentor others, creating a multiplier effect for Cardano adoption in Vietnam.

The 94% interest rate in advanced learning among course graduates demonstrates strong demand and positions Vietnam as an emerging hub for Cardano development talent.
